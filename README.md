# usage
##### install [python v313](https://www.python.org/downloads/)
##### install requirements.txt
## Windows
``` terminal
cd folder_location/
```
```python
pip install requirements.txt
```
##### run python project
```python
# windows
py main.py
```
## macosx & linux
``` terminal
cd ./folder_location/
```
```python
pip install requirements.txt
```
##### run python project
```python
# macosx & linux
python main.py
```
